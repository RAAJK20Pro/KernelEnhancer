About This Module

KernelEnhancer is a performance-optimized Magisk module designed for advanced Android users. It applies safe and effective systemless tweaks to maximize your device's potential.

Key Features

  • CPU Optimization: Governor tweaks and frequency control
  • GPU Boost: Reduced throttling and enhanced rendering
  • Memory Management: ZRAM, swap, and cache optimizations
  • I/O Scheduler: Improved storage performance
  • System Tweaks: VM, entropy, and kernel parameters
  • Automation: Applies at boot without root conflicts

Installation

Available as a Magisk module or standalone script:

Download Latest Release

Magisk Method:

Magisk Manager → Modules → Install from storage

Manual Method:

adb push KernelEnhancer.zip /sdcard

Compatibility

  • Android 9+ (Pie and above)
  • Rooted with Magisk 20.0+
  • Most ARM/ARM64 devices
