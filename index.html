KernelEnhancer

KernelEnhancer is a performance-optimized Magisk module designed for advanced Android users. It applies systemless tweaks to maximize your device's CPU, GPU, I/O, and RAM capabilities.

  • CPU Optimization: Governor tuning & responsiveness
  • GPU Boost: Disable throttling and increase frame stability
  • Memory Tweaks: ZRAM, swap, cache pressure
  • I/O Scheduler: Faster storage access and batching
  • System Tuning: VM, entropy, kernel logging & scheduling

GameEnhancer

GameEnhancer is an intelligent Magisk module that automatically detects when you launch a game and applies real-time performance boosts using CPU governor switching and thread priority tuning (renice + SCHED_FIFO).

  • Game Detection: Monitors running processes every 2 seconds
  • Real-time Prioritization: Applies renice & SCHED_FIFO to game PIDs
  • Governor Boost: Automatically switches to performance mode during gameplay
  • Auto-Revert: Restores defaults after game closes
  • Logging: Saves logs to both /data/local/tmp and /sdcard
